# Cookie 使用说明

本文档说明 VidkNot 项目中 Cookie 的获取和使用方式。

---

## 概述

VidkNot 使用 Cookie 来访问需要登录验证的视频平台。

### Cookie 的作用

| 平台 | 需要 Cookie | 说明 |
|------|-------------|------|
| YouTube | ✅ | 访问会员内容、年龄限制内容 |
| Bilibili | ✅ | 访问需要登录的内容 |
| TikTok | ✅ | 访问需要登录的内容 |
| 抖音 | ❌ | 不需要 |
| 小红书 | ⚠️ | 部分内容 |
| Instagram | ✅ | 需要登录 |
| Twitter/X | ✅ | 需要登录 |

---

## 获取 Cookie 的方式

### 方式一：浏览器扩展（推荐）

#### 安装步骤

1. 安装 Chrome 扩展 **Get cookies.txt LOCALLY**
   - 地址：https://chrome.google.com/webstore

2. 登录需要获取 Cookie 的网站
   - YouTube: https://youtube.com
   - Bilibili: https://bilibili.com

3. 点击扩展图标，点击 "Export"
   - 导出为 Netscape 格式

4. 保存到项目目录
   ```
   cookies/
   ├── youtube.txt
   ├── bilibili.txt
   └── tiktok.txt
   ```

#### 注意事项

- 每个网站需要单独导出
- Cookie 有有效期，过期后需重新导出
- 不要分享你的 Cookie 给他人

### 方式二：浏览器导出工具

#### Windows Chrome

```powershell
# 方法1: 使用浏览器内置导出
# 1. 打开 DevTools (F12)
# 2. Application → Cookies
# 3. 右键 → Export

# 方法2: 使用扩展（推荐）
# Get cookies.txt LOCALLY
```

#### macOS Chrome

```bash
# 使用扩展 Get cookies.txt LOCALLY
# 导出 Netscape 格式的 Cookie 文件
```

### 方式三：浏览器数据库复制

VidkNot 支持直接从浏览器数据库读取 Cookie：

```bash
# 确保浏览器已关闭
# VidkNot 会自动从 Chrome 读取 Cookie
yt-dlp --cookies-from-browser chrome "URL"
```

**注意**: Windows 上可能需要管理员权限。

## Cookie 文件格式

### Netscape 格式（推荐）

```bash
# youtube.txt 示例
# Netscape HTTP Cookie File
# This file was generated by Get cookies.txt LOCALLY

.youtube.com	TRUE	/	TRUE	1735689600	GOOGLE_ABIS	xxxxxxxxxx
.youtube.com	TRUE	/	FALSE	1735689600	LOGIN_INFO	xxxxxxxxxx
```

### JSON 格式

某些浏览器扩展导出为 JSON 格式，VidkNot 也支持。

---

## Cookie 存储位置

VidkNot 支持以下 Cookie 存储方式：

### 1. 项目目录

```
vidknot/
├── cookies/
│   ├── youtube.txt
│   ├── bilibili.txt
│   └── tiktok.txt
```

### 2. 环境变量指定

```bash
COOKIE_FILE=/path/to/cookies.txt
```

### 3. 临时文件

VidkNot 会尝试创建临时 Cookie 文件，使用后自动删除。

---

## Cookie 安全说明

### ⚠️ 重要警告

1. **Cookie 包含登录凭证**
   - 不要将 Cookie 文件上传到 Git
   - 不要分享给他人

2. **Cookie 有效期**
   - 通常 7-30 天
   - 过期后需要重新导出

3. **敏感操作**
   - 定期清理不使用的 Cookie
   - 发现异常立即撤销并重新登录

### 安全建议

```bash
# .gitignore 已排除 Cookie 目录
# 确保 cookies/ 目录在 .gitignore 中

# 定期更换 Cookie
# 使用完后关闭浏览器
```

## Cookie 与隐私

### VidkNot 不会

- ❌ 上传 Cookie 到任何服务器
- ❌ 存储 Cookie 在云端
- ❌ 分享 Cookie 给第三方

### VidkNot 只会

- ✅ 读取本地 Cookie 文件
- ✅ 临时使用 Cookie 下载视频
- ✅ 使用后自动清理临时文件

---

## 故障排除

### Cookie 过期

```
ERROR: Login required. Please refresh the cookie
```

**解决方案**: 重新导出 Cookie

### Cookie 无效

```
ERROR: This video is not available
```

**解决方案**: 检查 Cookie 是否为该平台的正确格式

### 权限不足

```
ERROR: Cannot read cookies database
```

**解决方案**:
- Windows: 以管理员身份运行
- macOS: 确保有文件读取权限
